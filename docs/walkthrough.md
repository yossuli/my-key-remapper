# Verification Guide: Windows Key Remapper MVP

このプロジェクトは Linux 環境で開発されていますが、Windows 用バイナリ（.exe）の生成には `wine` が必要なため、現在の開発環境ではビルドを完了できませんでした。
そのため、お客様の Windows 環境にてビルドを行っていただく必要があります。

## 必要なもの

- Windows PC
- Node.js (v18 以上推奨)

## 手順

1. **プロジェクトの取得**:
   このプロジェクトのファイル一式を Windows マシンにダウンロード（または git pull）してください。

2. **依存関係のインストール**:
   コマンドプロンプトまたは PowerShell でプロジェクトフォルダを開き、以下を実行します。

   ```bash
   npm install
   ```

   > これにより、Windows 用の `koffi` バイナリなどが自動的にセットアップされます。

3. **ビルドの実行**:
   以下のコマンドを実行して、Windows 用インストーラー（または実行ファイル）を作成します。

   ```bash
   npm run build:win
   ```

4. **生成物の確認**:
   `dist` フォルダが作成され、その中に `win-unpacked` フォルダ（または Zip ファイル）が含まれています。

5. **実行と動作確認**:
   - `dist/win-unpacked/WindowsKeyRemapper.exe` を実行します。
   - アプリ起動後、キー入力を行い、画面上のログやデバッグツールでフックが機能しているか確認してください。
   - ※ 本番ビルドでは `console.log` は見えない場合があるため、画面内のログ表示（"Key Pressed: ..."）を確認してください。

## トラブルシューティング

- **ビルドエラー**: `npm install` 時にネットワークエラー等が発生していないか確認してください。
- **フックが動作しない**: 管理者権限が必要な場合がありますが、まずは通常起動で試してください。`User32.dll` のロードに失敗している場合は、Windows Defender 等が干渉している可能性も考慮してください。
